# TD3-FrequencyAnalizer

Proyecto de KiCad para el Analizador de Frecuencias (TD3).

## Descripción del Proyecto

Este es un proyecto de diseño de PCB desarrollado en KiCad 10.0.4.

## Estructura del Proyecto

*   **`TD3-FrequencyAnalizer.kicad_pro`**: Archivo de proyecto de KiCad.
*   **`TD3-FrequencyAnalizer.kicad_sch`**: Esquema del circuito.
*   **`TD3-FrequencyAnalizer.kicad_pcb`**: Diseño del circuito impreso (PCB).
*   **`.gitignore`**: Exclusiones de Git para evitar subir archivos temporales y copias de seguridad de KiCad.
